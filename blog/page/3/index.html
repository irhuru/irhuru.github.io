Phonetic transcription systems you should know

What is phonetic transcription? Phonetic transcription is the use of phonetic symbols to represent speech. We could say it's a way to visually represent speech sounds, which is important because orthography often doesn't represent pronunciation accurately. Each phonetic symbol represents a sound in a spoken utterance. However, which phonetic symbol we use depends on the phonetic transcription system we follow.
