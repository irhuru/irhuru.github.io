Haiku Generator

This project produces a random haiku generator.

By using texts coming from actual books, a Python script automatically extracts words and outputs a haiku (a text that follows the 5-7-5 syllable rule).

Texts were taken from Project Gutenberg, an online library of more than 60,000 books. Among the Python libraries used for this project were SpaCy and SyllaPy.

You can learn more about this project by reading this blog post or by checking out the code on Github.

Experimental Linguistics with Ibex Farm

Ibex Farm üêê is an online resource that allows you to create psycholinguistic experiments that you can then send to participants. Ibex Farm runs using JavaScript, HTML, and CSS üíª.

The cool thing about it is that it is highly customizable and it allows you to measure reaction times (RTs) and to randomize items, among other things. You can check out the Ibex Farm website here and its documentation here.

I have used Ibex Farm to design picture-matching tasks where participants read sentences and had to choose the picture that best matched each sentence. All pictures were hosted online so that they could be linked to the experimental items and displayed on any device. I recommend using Imgur for the pictures.

You can find all the code here.

Heritage Language Writing

Heritage speakers are a specific type of bilingual speakers who grew up exposed to a home language different from the one spoken by majority in society. When heritage speakers start school, they learn the majority language which, in most cases, ends up being their dominant language üó£Ô∏è.

In the context of the United States üá∫üá∏, some colleges offer language courses specifically designed for heritage speakers, whereas others offer only mixed language courses (i.e., both for heritage and for non-heritage speakers). It is still unclear whether heritage-only language courses are better than mixed language courses.

This project compares the written assignments produced by a group of students enrolled in a heritage-only Spanish language course with those produced by a group of students in a mixed language course to understand the impact of instruction on students’ writing skills üìù.

All text files were imported into R and analyzed using natural language processing (NLP) tools and techniques in order to explore different areas:

  • Lexical density
  • Lexical diversity
  • Syntactic complexity
  • Lexical sophistication

Statistical analyses were conducted to examine how these areas developed throughout the course of a semester.

You can find the code here.

Bilinguals in the Midwest (BILinMID) Corpus

This corpus documents the Spanish and English spoken by bilinguals living in the Midwest of the United States. The corpus comprises written transcriptions of the Little Red Riding Hood fairy tale as narrated by several bilingual speakers. These stories were elicited using a children’s book (containing only images, not text) üìñ.

Bilingual speakers were either born in the United States or in a Spanish-speaking country (e.g., Mexico, Spain…), and they were fluent in both English and Spanish, though their level of proficiency differed based on individual variables, such as length of residence in the United States, country of birth, etc.

This is the first corpus ever to document the Spanish and English spoken in this region of the United States. The process of corpus creation is described in a paper that will be published soon (stay tuned!).
