Experimental Linguistics with Ibex Farm

Ibex Farm 🐐 is an online resource that allows you to create psycholinguistic experiments that you can then send to participants. Ibex Farm runs using JavaScript, HTML, and CSS 💻.

The cool thing about it is that it is highly customizable and it allows you to measure reaction times (RTs) and to randomize items, among other things. You can check out the Ibex Farm website here and its documentation here.

I have used Ibex Farm to design picture-matching tasks where participants read sentences and had to choose the picture that best matched each sentence. All pictures were hosted online so that they could be linked to the experimental items and displayed on any device. I recommend using Imgur for the pictures.

You can find all the code here.

The Languages of Frida Kahlo

This project compares the English and Spanish subtitles from the movie Frida (2002) 🎥. Subtitle files were imported into R and examined using several natural language processing (NLP) tools and techniques, such as:

  • Tokenization
  • Bigrams
  • Sentiment analysis

Results were also plotted for a better interpretation 📊📈. Some of the packages used include:

  • tidyverse
  • tidytext
  • stringi
  • igraph
  • ggraph
  • syuzhet

You can find the code here.

Heritage Language Writing

Heritage speakers are a specific type of bilingual speakers who grew up exposed to a home language different from the one spoken by majority in society. When heritage speakers start school, they learn the majority language which, in most cases, ends up being their dominant language 🗣️.

In the context of the United States 🇺🇸, some colleges offer language courses specifically designed for heritage speakers, whereas others offer only mixed language courses (i.e., both for heritage and for non-heritage speakers). It is still unclear whether heritage-only language courses are better than mixed language courses.

This project compares the written assignments produced by a group of students enrolled in a heritage-only Spanish language course with those produced by a group of students in a mixed language course to understand the impact of instruction on students’ writing skills 📝.

All text files were imported into R and analyzed using natural language processing (NLP) tools and techniques in order to explore different areas:

  • Lexical density
  • Lexical diversity
  • Syntactic complexity
  • Lexical sophistication

Statistical analyses were conducted to examine how these areas developed throughout the course of a semester.
